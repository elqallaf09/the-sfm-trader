# تثبيت the-sfm trader على الآيفون بشكل خاص

هذه الطريقة تحول الموقع إلى تطبيق خاص على الآيفون بدون App Store.

## الخطوات

1. شغل التطبيق من الكمبيوتر:
   ```powershell
   cd "C:\Users\User\the-sfm trader"
   npm start
   ```

2. تأكد أن الآيفون والكمبيوتر على نفس شبكة الواي فاي.

3. افتح Safari في الآيفون واكتب رابط جهازك:
   ```text
   http://192.168.255.180:4173/
   ```

4. اضغط زر المشاركة في Safari.

5. اختر:
   ```text
   Add to Home Screen
   ```

6. خل الاسم:
   ```text
   the-sfm trader
   ```

7. اضغط Add.

بعدها يظهر التطبيق على شاشة الآيفون ويشتغل كأنه تطبيق مستقل.

## مهم

- لازم الكمبيوتر يكون شغال والسيرفر مفتوح إذا تستخدم الرابط المحلي.
- إذا تبيه يشتغل من أي مكان خارج البيت، تحتاج استضافة خاصة أو VPN مثل Tailscale.
- هذه نسخة خاصة لك وليست منشورة في App Store.
