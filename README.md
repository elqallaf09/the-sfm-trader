# the-sfm trader

powered by M.ALQ

تطبيق ويب عربي يعرض توصيات آلية للأسواق: فوركس، الكويت، السعودية، الخليج، السوق الأمريكي، أوروبا، آسيا، وكل العالم.

## التشغيل

```powershell
cd "C:\Users\User\the-sfm trader"
npm start
```

ثم افتح:

```text
http://localhost:4173
```

## ملاحظات مهمة

- التوصيات تعليمية وليست نصيحة مالية.
- الواجهة تطلب تحديثا كل ثانية، لكن تغير السعر الحقيقي يعتمد على السوق ومزود البيانات.
- بيانات الأسهم تأتي من Yahoo Finance عبر endpoint عام، لذلك قد تكون بعض أسواق الخليج ناقصة أو متأخرة.
- إذا كان المنفذ 4173 مشغولا، يحاول السيرفر تشغيل نفسه على المنفذ التالي مثل 4174 ويطبع الرابط في PowerShell.
- الواجهة تحتوي على خلفية حركة أسواق، شريط أسعار متحرك، بحث، فلترة، فرز، ملخص أفضل فرصة، ورسوم مصغرة لكل سهم.
- عند اختيار السوق الأمريكي تظهر خانة خاصة لتوقعات الصعود خلال شهر، شهرين، و3 شهور، مع هدف السعر مثل "الصعود إلى 12.00 USD خلال شهرين".
- في بطاقات السوق الأمريكي يظهر وسم "مطابق للشريعة" تحت شارة شراء/بيع للأسهم المصنفة محليا كمطابقة، ويجب تحديث التصنيف دوريا من مزود فحص شرعي موثوق.
- تم تجهيز تنبيهات ذكية: شراء + مطابق للشريعة + ثقة 70% أو أكثر.
- تم تجهيز فلتر شرعي داخل التوصيات: كل التصنيفات، مطابق، غير مطابق، مختلف عليه.
- كل بطاقة تعرض مؤشر مخاطرة ونتيجة اختبار خلفي Backtest مبسط على آخر بيانات متاحة.
- يمكن تبديل مزود البيانات عبر متغيرات البيئة الموضحة في `.env.example`: Yahoo افتراضي، أو Finnhub/Alpha Vantage عند توفر API key.
- لإضافة رموز جديدة، عدل ملف:

```text
C:\Users\User\the-sfm trader\src\markets.mjs
```

## تطوير لاحق مقترح

- ربط مزود بيانات احترافي مثل Polygon, Finnhub, Twelve Data, Refinitiv, Bloomberg أو مزود البورصة الرسمي.
- إضافة WebSocket للتحديث اللحظي الحقيقي.
- إضافة تسجيل دخول، تنبيهات، سجل توصيات، وإدارة محفظة.
- إضافة نموذج تعلم آلي مخصص بعد توفر بيانات تاريخية نظيفة وكافية.
